# English Speaking Practice Tool

一个优雅的英语口语练习复盘工具，帮助你高效整理和复习口语练习内容。

## ✨ 功能特点

### 📝 智能标记系统
- **背景颜色标记**：5种颜色分类标记（错误/待改进/正确/重点/生词）
- **文字颜色标记**：5种颜色突出显示重要内容
- **即点即用**：选中文字后点击颜色立即应用，无需额外操作

### 💬 内联批注功能
- **行内批注气泡**：批注显示在文字下方，不占用额外行空间
- **回车应用**：输入批注后按回车键即可添加，操作流畅
- **点击编辑**：点击批注气泡可快速编辑内容
- **智能层级**：批注重叠时点击自动置顶

### 🔊 发音助手
- **内置发音**：选中单词或句子，点击发音按钮即可听到标准美式发音
- **零配置**：使用浏览器内置语音合成，无需API或网络
- **即时反馈**：点击即读，学习更高效

### 📚 笔记管理
- **多笔记支持**：创建多个练习笔记，分类管理
- **文件夹组织**：支持创建嵌套文件夹，拖放笔记进行分类
- **自动保存**：编辑内容实时自动保存到本地
- **复盘记录**：标记复盘次数和时间，追踪学习进度
- **自动恢复**：下次打开自动恢复到上次退出的笔记

### ✏️ 改进版本对照
- **双编辑器**：上方编辑原始文稿+标记，下方输入改进版本
- **对比学习**：方便查看原文和改进版，加深记忆
- **独立编辑**：两个编辑器独立工作，互不干扰

### 🎨 优雅的用户体验
- **左下角提示**：所有提示信息以toast形式显示，不打断操作
- **悬浮工具栏**：选中文字即显示工具栏，功能一步到位
- **保持选中**：颜色应用后保持选中状态，可连续调整
- **极简设计**：淡雅配色，专注内容，减少视觉干扰

## 🚀 快速开始

1. **打开工具**
   - 双击 `index.html` 即可在浏览器中打开
   - 无需安装，纯前端应用

2. **基本使用**
   - 在编辑器中输入或粘贴你的口语练习文稿
   - 选中需要标记的文字，点击背景颜色或文字颜色
   - 输入批注后按回车键添加
   - 点击🔊发音按钮听单词读音

3. **数据存储**
   - 所有数据保存在浏览器本地存储（localStorage）
   - 数据不会上传到任何服务器
   - **支持导出/导入**：可将数据导出为JSON文件备份
   - 定期导出数据到 `data` 文件夹，防止数据丢失

## 💡 使用技巧

### 颜色标记建议
- 🔴 **红色**：错误的表达
- 🟠 **橙色**：需要改进的地方
- 🟢 **绿色**：正确的表达
- 🟣 **紫色**：重点表达或高级词汇
- 🔵 **蓝色**：生词或短语

### 批注技巧
- 记录正确的表达方式
- 标注语法要点
- 记录同义词替换
- 注明发音难点

### 文件夹管理技巧
- 点击 **+ 创建文件夹** 按钮创建根文件夹
- 悬停在文件夹上点击 **+** 可创建子文件夹
- 直接拖动笔记到文件夹上进行分类
- 点击文件夹名称或 **▸** 图标展开/折叠

### 数据备份建议
- 定期点击 **💾 导出** 按钮备份数据
- 将导出的JSON文件保存到 `data` 文件夹
- 需要恢复时点击 **📂 导入** 按钮
- 建议每周或完成重要笔记后导出一次

### 工作流程建议
1. 粘贴原始练习文稿到上方编辑器
2. 用不同颜色标记错误、重点等
3. 添加批注说明问题和改进方向
4. 在下方编辑器输入改进后的版本
5. 对比学习，加深印象
6. 定期复盘，巩固知识
7. 使用文件夹整理笔记，定期导出备份
