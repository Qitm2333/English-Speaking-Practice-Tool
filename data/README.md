# 数据备份说明

## 使用方法

### 导出数据
1. 点击左侧边栏的 **💾 导出** 按钮
2. 浏览器会自动下载一个 JSON 文件（格式：`notes-backup-YYYY-MM-DD.json`）
3. 将下载的文件保存到此 `data` 文件夹中

### 导入数据
1. 点击左侧边栏的 **📂 导入** 按钮
2. 选择之前导出的 JSON 文件
3. 确认导入后，所有笔记和文件夹将被恢复

## 为什么需要导出数据？

目前数据保存在浏览器的 localStorage 中，虽然方便，但有以下限制：
- **清理浏览器缓存会导致数据丢失**
- 无法跨设备同步
- 没有历史版本

通过定期导出数据到此文件夹，你可以：
✅ 防止数据丢失
✅ 创建备份版本
✅ 在不同设备间迁移数据

## 建议

建议每周或每次做完重要笔记后导出一次数据，以确保数据安全。
